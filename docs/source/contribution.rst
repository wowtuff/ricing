Contribution
============

Contributing areas
------------------

Useful contribution areas for Agentic Ricing include:

- more Linux desktop and WM/DE integrations
- better preview and diff views in the browser UI
- safer config patching flows
- richer package-manager support
- support for more ricing-specific tools
- better reproducibility and test coverage

How to contribute
-----------------

1. Fork the repository
2. Create a branch
3. Implement the feature or fix
4. Test the workflow
5. Submit a pull request
